Schemata for Verifiable Credentials

Expressing verifiable information on the Web

Welcome to credentialschema.nz

The purpose of this site is to provide the easiest way to create, maintain, and promote schemata for use with the W3C Verifiable Credentials Data Model, you can check out the what's available below.

This schema is great for organisational office, site or team inductions.
